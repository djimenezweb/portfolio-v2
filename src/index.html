Darkroom Finder

Un buscador de laboratorios de fotografía hecho con Next.js, React, TypeScript y Tailwind CSS. Incluye autenticación con NextAuth, búsquedas facetadas con MongoDB, paginación, validación de formularios con Zod, almacenamiento de archivos en Cloudinary y visualización de mapas con MapLibre. Bidder

Una aplicación de subastas hecha con React y Firebase. Los usuarios pueden darse de alta e iniciar sesión, publicar anuncios con fotos y pujar en otras subastas. La autenticación de usuarios y la base de datos está hecha en Firebase. También se han usado las librerías React Router para las rutas, React Hook Form para la validación de los formularios, Styled Components para los estilos y Framer Motion para las animaciones.
